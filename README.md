A WordPress plugin for ghost kitchens, virtual restaurant brands, and food delivery operators managing multi-channel order flow. Aggregates orders from DoorDash, Uber Eats, and Grubhub APIs into a unified kitchen display view, manages prep time estimates by menu category, tracks order accuracy rates by delivery partner, generates daily revenue reconciliation reports by channel, and sends automated low-inventory alerts to the procurement team when popular items run low.

== Description ==

A WordPress plugin for ghost kitchens, virtual restaurant brands, and food delivery operators managing multi-channel order flow. Aggregates orders from DoorDash, Uber Eats, and Grubhub APIs into a unified kitchen display view, manages prep time estimates by menu category, tracks order accuracy rates by delivery partner, generates daily revenue reconciliation reports by channel, and sends automated low-inventory alerts to the procurement team when popular items run low.

**Features:**

* Core functionality

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate via WordPress admin > Plugins
3. Configure via Settings > Food Delivery and Ghost Kitchen Order Management WordPress Plugin
